# StudentPulse
Secure PHP/MySQL student events portal for Internet Programming II.

## Install
1. Copy this folder to `C:\xampp\htdocs\`.
2. Start Apache and MySQL.
3. Import `database.sql` in phpMyAdmin.
4. Open `http://localhost/StudentPulse/`.

## Demo accounts
- Admin: admin@studentpulse.test / password
- Student: student@studentpulse.test / password

## Login paths
- Student login: `/login.php`
- Admin login: `/admin/login.php`

## Security
mysqli prepared statements, password hashing, server validation, output escaping, CSRF tokens, session regeneration, secure logout and separate student/admin role checks.
